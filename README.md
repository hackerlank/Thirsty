## Installation

下载构建工具[premake5](http://premake.github.io/download.html).

* Windows 7 x64，双击`msvc2013.bat`生成Visual C++工程文件
* Ubuntu 14.04 x64，在Terminal里执行`premake5 gmake && make config=release`
